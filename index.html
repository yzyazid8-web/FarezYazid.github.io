PROJET 01

Reconnaissance d'Activité Humaine (HAR)

Développement d'un pipeline de classification sur le dataset HAR. Utilisation de l'algorithme LLE (Locally Linear Embedding) pour la réduction de dimensionnalité non-linéaire, couplé à un modèle SVM.

Focus : Réduction de dimensionnalité non-linéaire, Machine Learning supervisé, interprétation des performances.

Classification Python SVM LLE
→ GitHub

PROJET 02

Analyse de Fiabilité & Simulations Monte Carlo

Projet académique axé sur l'analyse de fiabilité industrielle. Modélisation des défaillances de composants via la distribution de Weibull et des inspections via une loi Gaussienne avec simulations Monte Carlo.

Focus : Modélisation probabiliste, méthodes stochastiques, simulation numérique.

Probabilités Python Monte Carlo Weibull
→ GitHub

PROJET 03

Modélisation de la Mortalité & Analyse de Survie

Analyse de données démographiques via le modèle de Lee-Carter et la loi de Gompertz-Makeham (package StMoMo). Mise en pratique avancée des estimateurs de Kaplan-Meier et modèles de Cox.

Focus : Analyse de survie, statistiques appliquées, modélisation mathématique sous R.

Analyse de Survie R StMoMo Cox Model
→ GitHub

PROJET 04

Analyse Topologique de Données (TDA)

Exploration de concepts complexes en TDA (complexes de Vietoris-Rips, diagrammes de persistance) pour extraire des caractéristiques topologiques de jeux de données, en utilisant la bibliothèque giotto-tda.

Focus : Topologie algébrique appliquée, extraction de features complexes.

TDA Python giotto-tda
